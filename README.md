# Data Jobs Dashboard w/ Power BI

<img src="images/Project1_Dashboard_Overview.gif">

## Introduction

This dashboard was built for **Job Seekers, Career Transitioners, and Data Professionals** to solve a common challenge: job market insights are often scattered, inconsistent, and difficult to interpret.

Using a real-world dataset of 2024 data job postings ‚Äî including job titles, salaries, locations, and employment types ‚Äî this project consolidates key insights into a single, interactive analytics experience. The goal is to provide a clear, data-driven view of market demand, compensation trends, and geographic hiring patterns.

## Dashboard Overview

*The report is divided into two primary pages to provide both a strategic overview and a detailed analytical breakdown.*

### Page 1: High-Level Market Overview

![Data Jobs Dashboard Page 1](images/Project1_Dashboard_Page1.gif)


This page serves as a comprehensive snapshot of the data job market. It highlights key KPIs such as total job count, median salaries, and top-paying roles. Users can analyze hiring trends throughout 2024, compare salary levels across job titles, and explore global job distribution ‚Äî all within a single, interactive view.

---

### Page 2: Job Title Drill-Through Analysis

![Data Jobs Dashboard Page 2](images/Project1_Dashboard_Page2.gif)


This page provides a focused, role-specific deep dive. By drilling through from the main dashboard, users can analyze:

- Salary ranges and median compensation
- Work-from-home availability
- Health insurance benefits
- Employment type breakdown
- Job posting platforms
- Geographic distribution for the selected role

This feature demonstrates advanced interactivity and contextual filtering within Power BI.
